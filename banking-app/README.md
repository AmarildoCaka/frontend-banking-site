# 💳 Banking Dashboard App

A modern and responsive banking web application built with **React**, **TypeScript**, and **Vite**.  
This app allows users to manage **deposits, withdrawals, bill payments, cards**, and **transaction history** through an intuitive dashboard interface.

It demonstrates **frontend architecture**, **state management**, and **financial data handling** in a real-world scenario.

---

## 🚀 Features

- User dashboard overview with account summary
- Deposit & withdraw money
- Bill management and bill payments
- Card management interface
- Transaction history tracking
- Global state management with **Zustand**
- Persistent data using **localStorage**
- Form validation and error handling
- Fully responsive design (desktop & mobile)
- Reusable, component-based architecture

---

## 🛠 Tech Stack

- **React** – Frontend library  
- **TypeScript** – Type safety and developer tooling  
- **Vite** – Development build tool with fast refresh  
- **Tailwind CSS** – Styling and responsive design  
- **Zustand** – Global state management  
- **React Router** – Navigation  
- **LocalStorage** – Data persistence  
- **Lucide / FontAwesome** – Icons
